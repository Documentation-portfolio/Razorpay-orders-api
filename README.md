# Razorpay Orders API Documentation
This repository contains my documentation for the **Razorpay Orders API**. I wrote api-documentation.md as part of my developer documentation and API writing portfolio. This documentation is designed to help developers integrate payment flows into their applications quickly and accurately.

## About the Razorpay Orders API
The Razorpay Orders API helps developers create and manage payment orders, retrieve order details, and fetch all payments associated with an order. It uses **Basic Authentication** with a Key ID and Key Secret, and supports JSON-based request/response workflows.

## What api-documentation.md Covers
- Basic Auth configuration
- How to create an order
- How to fetch orders with query filters
- How to fetch payments for an order
- How to capture a payment
- How to create a refund
- Request body and parameter reference
- Response schemas
- Curl examples
- Common error responses and fixes
